Background

Kernel Density Visualization (KDV) is a commonly-used visualization tool to explore and discover patterns from a dataset, which has been extensively used in a wide range of applications, including ecological modeling, crime hotspot detection, traffic accident hotspot detection and disease outbreak detection. In these applications, domain experts need to generate multiple KDVs, given a set of L bandwidth values. However, generating a single KDV is time-consuming, let alone to generate multiple KDVs. In this project, we develop a Share-and-Aggregate Bandwidth Exploration Framework, called SAFE, which can efficiently generate multiple KDVs for these L bandwidth values.
